支付宝账单

如何获取支付宝账单？

  1. 打开支付宝 App -> 我的 -> 账单
  2. 右上角 ... -> 开具交易流水证明 -> 用于个人对账 -> 申请
  3. 自定义时间范围（最长为一年） -> 填写邮箱 -> 下载账单
  4. 申请记录中找到解压密码 -> 解压下载的文件，获取 CSV 文件
  5. 按年份重命名为【alipay_record_2024.csv】格式

微信账单

如何获取微信账单？

  1. 打开微信 App -> 我 -> 服务 -> 钱包 -> 账单
  2. 点击右上角 [...] -> 账单下载 -> 用于个人对账
  3. 选择接收方式（微信/邮箱）、账单时间 -> 下一步
  4. 方式一（推荐）：刷脸验证 -> 微信消息直接下载 XLSX 文件
  5. 方式二：输入邮箱 -> 刷脸 -> 邮箱接收 (密码在微信消息中)
